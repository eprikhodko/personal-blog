Higher-Order Components (Компоненты высшего порядка, HOC) - это способ, позволяющий повторно использовать код компонентов. <br />
Проще говоря, компонент высшего порядка - это функция, которая принимает компонент, и возвращает новый компонент.

```js
const EnhancedComponent = higherOrderComponent(WrappedComponent)
```
